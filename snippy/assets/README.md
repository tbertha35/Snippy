# Snippy Assets

This directory holds static assets shipped with the package:

- `icon.png` — the system tray / app icon (Phase 1 will use it)
- `sounds/` — capture confirmation sounds (Phase 1)
- `themes/` — bundled `.qss` themes (Phase 2)

The actual binary files are intentionally not committed in this scaffold —
they will be added when the corresponding features land. See `ROADMAP.md`.
